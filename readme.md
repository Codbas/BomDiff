An online tool to compare two CSV files<br>
https://bomdiff.com

## Changelog
11/6/2025 - Added Change column. This column will display the quantity of item changes between CSV1 and CSV2.
<br><br>
## About
Compare two CSV files based on column headers<br><br>
### Required Headers

**Item**<br>
"item", "component name", or "part"<br>

**Quantity**<br>
"quantity", or "quantity per top level assembly"<br>

**Description**<br>
"description"
